# Desafio Banco em Java

Projeto simples desenvolvido em Java para praticar lógica de programação, estruturas condicionais, laços de repetição e entrada de dados pelo terminal.

## Funcionalidades

- Consultar saldo
- Receber valor
- Transferir valor
- Validar saldo insuficiente
- Encerrar o programa

## Tecnologias utilizadas

- Java
- IntelliJ IDEA
- Git e GitHub

## Conceitos praticados

- Variáveis
- Scanner
- Estrutura `while`
- Estrutura `switch case`
- Condicionais `if/else`
- Operações matemáticas básicas
- Controle de fluxo com `return`
